# My JavaScript Learning Journey

This repository documents my self-learning journey in JavaScript.

## Learning Resources

I am learning JavaScript using the following resources:

- **The main resource is the JavaScript handbook provided by the instructor.**
- YouTube tutorials

My goal is not only to watch tutorials or read explanations. I want to understand JavaScript, practice every concept, solve problems independently, build projects and document my progress on GitHub.

---

# How I Learn JavaScript

For every topic, I follow this learning process:

1. Read a basic explanation of the topic.
2. Watch one or two relevant YouTube tutorials.
3. Search Google when I need another explanation.
4. Check official documentation for correct syntax and behaviour.
5. Type every code example myself.
6. Predict the output before running the code.
7. Change the example values and observe the result.
8. Write my own small examples.
9. Complete practice problems without copying solutions.
10. Use AI to explain difficult concepts and errors.
11. Review every AI-generated answer carefully.
12. Build a small project after completing a group of topics.
13. Push my notes, exercises and projects to GitHub.
14. Revise previous topics regularly.

---

# My Rules for Using AI

I use AI as a learning assistant, not as a replacement for my own thinking.

- I try to solve a problem myself first.
- I ask AI to explain concepts in simple language.
- I ask AI to explain errors instead of only fixing them.
- I read and understand every generated line of code.
- I test all AI-generated code.
- I change and improve the code myself.
- I do not submit code that I cannot explain.
- I do not share passwords, API keys or private information.
- I keep a record of important AI assistance used in projects.
